NODEJS
1. Instalar nodejs pelo cmd 
2. Crear una pasta do projeto
3. Ejecutar comando npm init -y (instalar packge.json)
4. instalar framework Express, control de rutas. 
	Comando: npm install express
6.Instalar insomnia
7. instalar nodemon, comando: npm install nodemon -D.
8. Configuração package.json. Scripts "start": "nodemon index.js".
9. Instalar knexjs.org
             Comandos: npm install knex --save
             Comandos: npm install sqlite3
	     Comandos: npx knex init
10. Reorganizar o codigo en carpetas
11. cambiar la ruta de nodemon en packege.json
12. criar as rutas
13. criar a pasta banco de dados e configurar rota da banco em knexfile.js 
em flename
14. Identificar as entidades e funcionalidades.
15. Criar migrations: 
  15A. Criar pasta Migrations.
  15b. Configurar route migration em knexfile.js.
  15c. Ejecutar comando: npx knex migrate:make create-ongs
  15d. useNullAsDefault: true
  15e. create table 
  15f. Correr a migrations comando: npx knex migrate:latest
16.npm install cors
